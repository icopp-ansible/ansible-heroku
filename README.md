# ansible-heroku

Install the latest stable Heroku CLI via Homebrew.

## Dependencies

* [icopp.homebrew](https://github.com/icopp/ansible-homebrew)
* [icopp.ruby](https://github.com/icopp/ansible-ruby)

## Example Playbook

```
  - hosts: all
    roles:
      - role: icopp.heroku
```
